Malengo Solution .

Malengo is a Ruby on Rails Application designed and built from scratch by a team of three software engineers. It provides an interface where shop owners can monitor and manage their businesses

The client is responsible for feeding data into the system and it's later analyzed and presented visually for easy assessment.

Inspiration

I have always been intrigued by enterprise software and with some background in technology, the best way was to build software that I can use to run and manage my business.


This was also a good opportunity to work alongside mentors(teammates) on how to build a production-grade application.


Project Purpose and Goal

The main objective is to help clients know if they are indeed running a sustainable business.

This is done by tracking stock, sales, and expenses, calculating profits, and prompting customers if a product is out of stock.

    Handle Stock Outs

    We had a ticket on handling stockouts. It involved modeling and structuring data in First In First Out(FIFO).

    I tackled the ticket, wrapped the code in a Rails Concern, and re-used it across the application.

Lesson Learned

There are many ways to solve a problem. Pair programming improved my problem-solving skills by borrowing some techniques from my peers.


There were occurrences of code repetition. With knowledge of design patterns and principles, we refactored our code.


Software is an ongoing process, but we have deadlines. I mastered the art of prioritizing, owning tickets, and how to focus on the project scope.
